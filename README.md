# career-search
A prompt-driven career operations system for scanning roles, optimizing resumes, and managing applications with a product-minded workflow.

## Goal
Target senior-level product design roles with strong alignment to:
- SaaS
- AI-native or AI-forward products
- workflow-heavy applications
- design systems
- data-dense interfaces
- developer-adjacent or platform experiences

## Focus
- Senior Product Designer
- Lead Product Designer
- Staff Product Designer
- Senior UI/UX Designer
- Design Systems Lead
- Product Design Manager, IC leaning

## Compensation Target
- Minimum: $120K
- Ideal: $150K to $200K+

## Workflow
1. Run daily scans
2. Score roles
3. Tailor resume
4. Generate cover letter if needed
5. Apply
6. Track progress
7. Review weekly and refine

## Exclusions
- Security clearance roles
- contract-only roles
- junior or mid-level roles
- UX research-only roles
- low-budget agency work unless unusually strong fit
